Software de Gestion Administrativa Matrona
El proyecto Matrona tiene como objetivo construir un sistema que permita la administracion de una empresa que produce cerveza artesanal.

Instrucciones para instalacion y uso
Herramientas necesarias
1 Editor de codigo Visual Studio Code
2 Navegador web, se recomienda Google Chrome
3 Se requiere instalar extension Live Server en vs code 

Intrucciones de intalación y uso
1 Descomprimir el archivo zip
2 Ubicar la carpeta Proyecto Matrona
3 Abrir la carpeta directamente desde visual studio code, o arrastrar carpeta hasta el editor de codigo.
4 Seleccionar el archivo inicioSesion.html del proyecto y abrirlo con live server para visualizarlo en el navegador.
5 Introducir cualquier valor en nombre y contraseña y dar click en iniciar sesion para acceder al menu
6 La navegacion del proyecto entre paginas esta completa, desde la interfaz del menu podra desplazarce hacia las demas pantallas mediante el uso los botones de la barra de navegacion izquierda
7 La interfaz de clientes Matrona empiza desde catalogo de productos y tiene sus opciones, historialCliente.html y pedidosCliente.html, para salir dar click en cerrar sesion y llevara 
nuevamente a la pagina de inicioSesion.



Detalles a tener en Cuenta
El archivo tailwind.config.js es necesario para la configuracion del autocompletado de tailwind, es por eso
que es necesario tenerlo en ese lugar, es un problema de la version que seguranemte resolveran pronto.
